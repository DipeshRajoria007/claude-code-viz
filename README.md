# claude-code-viz

A local web dashboard for your Claude Code data (`~/.claude`).

```sh
npx claude-code-viz
```

Visualize your Claude Code usage — token & cost analytics, session browser with full conversation replay, activity heatmap, and tool analytics. 100% local and read-only: your data never leaves your machine.

> 🚧 Under active development — first release coming soon.
